# Customer-Analytics-Segmentation-RFM-Model-

Tech Stack: BigQuery (SQL), Power BI

Overview:
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis on 12 months of sales data to identify high-value and at-risk customers.

Steps:

Combined monthly sales data into a single dataset using SQL
Calculated RFM metrics and applied scoring using window functions
Segmented customers into categories like Champions, Loyal, and At Risk
Built an interactive Power BI dashboard for insights

Key Insights:

Identified high-value customers contributing most revenue
Detected low-engagement customers at risk of churn
